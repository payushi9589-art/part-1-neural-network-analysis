# Part 1: Neural Network Analysis

**Weights and Biases:** 

Weights determine feature importance. Biases act as an anchor, allowing the activation function to shift to fit the data.


**Activation Function:** 

It introduces non-linearity, allowing the network to learn complex patterns instead of just straight lines.


**Learning Rate:** 

If too high, the model overshoots and fails to learn. If too low, training takes forever or gets stuck.
